# 🛒 Home Appliance Store

پروژه فروشگاه اینترنتی لوازم خانگی با استفاده از فریم‌ورک **Django**

---

## 📌 معرفی پروژه

این پروژه یک فروشگاه اینترنتی ساده برای ارائه دانشگاهی است که با Django پیاده‌سازی شده است.
در این فروشگاه کاربران می‌توانند ثبت‌نام کنند، وارد حساب کاربری خود شوند، محصولات را مشاهده کنند، کالاها را به سبد خرید اضافه کنند و سفارش خود را ثبت نمایند.

این نسخه، نسخه اولیه پروژه است و همچنان در حال توسعه و تکمیل می‌باشد.

---

## 🚀 امکانات فعلی پروژه

* ثبت‌نام کاربر
* ورود و خروج کاربر
* پروفایل کاربری
* نمایش لیست محصولات
* نمایش جزئیات محصول
* دسته‌بندی محصولات
* سبد خرید
* شروع پیاده‌سازی ثبت سفارش
* پشتیبانی از چند محصول در هر سفارش با مدل `OrderItem`
* پنل مدیریت Django برای مدیریت محصولات، دسته‌بندی‌ها و سفارش‌ها

---

## 🧩 اپلیکیشن‌های پروژه

```text
homeappliance/     تنظیمات اصلی پروژه
core/              صفحات عمومی سایت
shop/              محصولات، دسته‌بندی‌ها، سفارش‌ها و امتیازدهی
cart/              سبد خرید
accounts/          ثبت‌نام، ورود، خروج و پروفایل کاربران
templates/         قالب‌های HTML
static/            فایل‌های CSS و JavaScript
```

## ⚙️ راه‌اندازی پروژه

ابتدا پروژه را دریافت کرده و وارد پوشه پروژه شوید:

```bash
cd "Project Name"
```

ساخت محیط مجازی:

```bash
python -m venv venv
```

فعال‌سازی محیط مجازی در Linux / macOS:

```bash
source venv/bin/activate
```

فعال‌سازی محیط مجازی در Windows:

```bash
venv\Scripts\activate
```

نصب وابستگی‌ها:

```bash
pip install -r requirements.txt
```

اجرای migrationها و ساخت دیتابیس:

```bash
python manage.py migrate
```

ساخت کاربر مدیر:

```bash
python manage.py createsuperuser
```

اجرای پروژه:

```bash
python manage.py runserver
```

سپس آدرس زیر را در مرورگر باز کنید:

```text
http://127.0.0.1:8000/
```

پنل مدیریت Django:

```text
http://127.0.0.1:8000/admin/
```
